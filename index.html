Anatomy of a container query

Codepen Link

Step 1: Set Containment on the parent


						.parent {
							container-type: inline-size;
						}

Step 2: Query the parent


						@container (max-width: 400px) {
							.card {
								// child styles when the parent is < 400px wide
							}
						}
					

						@container (max-width: 400px) {
							.card {
								grid-template-columns: 1fr;
							}
						}
					

						/* Default styles (wide) */
						.card {
							display: grid;
							grid-template-columns: 1fr 1fr;
						}

						/* Adjusted styles (narrow) */
						@container (max-width: 400px) {
							.card {
								grid-template-columns: 1fr;
							}
						}
					

						/* Default styles (narrow) */
						.card {
							display: grid;
							grid-template-columns: 1fr;
						}

						/* Adjusted styles (wide) */
						@container (min-width: 400px) {
							.card {
								grid-template-columns: 1fr 1fr;
							}
						}

Named Containers

Named containers


						.parent {
							container-type: inline-size;
							container-name: card-parent;
						}

container shorthand


						.parent {
							container: card-parent / inline-size;
						}
					

						.card {
							display: grid;
							grid-template-columns: 1fr;
						}

						/* More explicit with container-name */
						@container card-parent (min-width: 400px) {
							.card {
								grid-template-columns: 1fr 1fr;
							}
						}
